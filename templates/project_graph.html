Component Details
⚗️
Flask Backend
app.py — Core server
  • Routes: / /login /signup /detect-ai /detect-ai-video /uploads/<f>
  • Max upload: 500 MB per request
  • CORS via flask-cors
  • Jinja2 template rendering for all pages
  • Werkzeug secure_filename for safe I/O
👁️
Sightengine API
AI-Image Detection
  • Model: genai
  • Detects: Midjourney, DALL-E, Stable Diffusion
  • Returns ai_generated probability 0–1
  • Threshold: 0.7 for classification
  • Supported: PNG, JPG, JPEG, WEBP, GIF, BMP
  • 30s timeout with full HTTP error handling
🤗
Hugging Face API
AI-Video Frame Analysis
  • Model: umm-maybe/AI-image-detector
  • 3 frames at 25%, 50%, 75% of video duration
  • Each frame JPEG-encoded via OpenCV
  • Max score across frames = final verdict
  • Labels: artificial / LABEL_1 / FAKE
  • 60s timeout, graceful rate-limit fallback
🔥
Firebase Auth
Authentication layer
  • SDK: firebase_admin
  • Local: serviceAccountKey.json
  • Cloud: FIREBASE_CREDENTIALS JSON env var
  • Client-side auth via static/auth.js
  • Graceful fallback on SDK init failure
🧠
ML Training Pipeline
train_ensemble.py
  • Xception 299×299, EfficientNetB4 224×224, ResNet50 224×224
  • Phase 1: Warm-up — top layers only (5 epochs)
  • Phase 2: Fine-tune — all layers (configurable)
  • Social-media aug: JPEG noise, brightness, shift, zoom
  • Class-weight balancing via sklearn
  • Callbacks: ModelCheckpoint, EarlyStopping, ReduceLROnPlateau, TensorBoard
📦
Deployment
Render + Gunicorn
  • Platform: Render cloud PaaS
  • Process: gunicorn app:app via Procfile
  • Config: render.yaml with env group
  • Deploy deps: requirements-deploy.txt
  • Full deps: requirements.txt (48 packages)
